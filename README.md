# mcpelauncher-fix-icons

Небольшой скрипт-костыль для тех, кого бесит дефолтная серая иконка иксов (или значок Wayland) на панели задач при запуске Minecraft Bedrock Edition через Flatpak-лаунчер (`io.mrarm.mcpelauncher`) на Linux.

Проблема в том, что оконный менеджер часто не понимает, к какому приложению относится запущенное окно игры, и не может натянуть на него правильную картинку. 

Что делает этот фикс:
* Скрывает дубли и лишний мусор из стартового меню.
* Раскидывает правильные .desktop файлы с жестко прописанным классом окна (`StartupWMClass`) в `~/.local/share/applications/`.
* Закидывает кастомную SVG-иконку в стандартную директорию `hicolor`.
* Автоматом обновляет кэш иконок и сбрасывает кэш Плазмы, чтобы всё заработало без перезагрузок.

## Установка

Просто скопируй эту команду в терминал. Скрипт сам скачает актуальные файлы из этого репозитория и раскидает их по нужным папкам:

```bash
curl -sL [https://raw.githubusercontent.com/xesqaz/mcpelauncher-fix-icons/main/install.sh](https://raw.githubusercontent.com/xesqaz/mcpelauncher-fix-icons/main/install.sh) | bash
